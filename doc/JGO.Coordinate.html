Class: Coordinate

JGO. Coordinate

new Coordinate(i, j)

Create a helper class to create coordinates from (1,2) (zero-based), "ah" type of input. You can create a coordinate with no arguments, in which case it defaults to (0,0), or with one argument, in which case it tries to parse "ai" type of string coordinate, or with two arguments, (i,j). "J18" style coordinates depend on board size due to number running from bottom, so those need to be instantiated from JGO.Board.getCoordinate.
