addListener(event, callback)

Add an event listener to canvas (click) events. The callback will be called with 'this' referring to JGO.Canvas object, with coordinate and event as parameters. Supported event types are 'click', 'mousemove', and 'mouseout'. With 'mouseout', there is no coordinate parameter for callback.
Parameters:
Name Type Description
event String The event to listen to, e.g. 'click'.
callback function The callback.
